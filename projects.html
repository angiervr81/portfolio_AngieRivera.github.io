Scene Project

The Scene Assignment involves creating a visually composition using the HTML5 <canvas> element. It requires a background image, at least two foreground images, and text displaying the author's name and scene title. We learned HTML and JavaScript files, load images, and draw elements on the canvas. Flight Cost Calculator

This project is meant to create an interactive display that uses form inputs like text fields, checkboxes, and dropdowns to calculate and display the total cost based on user selections. A travel-themed background image enhances the visual appeal, aligning with the goal of combining math-based calculations with interactive, styled web forms

PAC Away

Interactive Pac-Man Canvas Scene is a fun and where users can interact with a customizable Pac-Man-themed scene. The canvas allows users to move a like alike Pac-Man character across the screen using a slider, toggle the visibility of Pac-Man fruits like cherries, strawberries, and oranges, and switch between different backgrounds such as a darkroom, city life, and beach . Additionally, the project includes sound effects that can be triggered by buttons, bringing an extra layer of interactivity. Using HTML, CSS, and JavaScript, this project provides an engaging way to explore the basics of web development and interactive design.
